## 2. Description of Service

MergeBetter is a Jira Cloud app that allows Jira administrators to define merge policies — sets of JQL-based rules that validate whether a Jira issue meets specific criteria before a GitHub pull request is allowed to merge.

The App consists of:
- A **Jira project settings UI** for configuring merge policies
- A **webhook endpoint** that GitHub Actions workflows call to check pull requests against configured policies
